# Variables

Variables are used to store data in DMJScript.

---

## String Variables

Strings store text.

```dmj
var name = "Durvesh"

show name
```

Output:

```text
Durvesh
```

---

## Integer Variables

Integers store whole numbers.

```dmj
var age = 17

show age
```

Output:

```text
17
```

---

## Decimal Variables

Decimals store floating-point numbers.

```dmj
var voltage = 4.75

show voltage
```

Output:

```text
4.75
```

---

## Multiple Variables

```dmj
var name = "Durvesh"
var age = 17
var voltage = 4.75

show name
show age
show voltage
```

Output:

```text
Durvesh
17
4.75
```

---

## Variable Rules

- Variable names are case-sensitive.
- Variable names should begin with a letter.
- Variable names may contain letters, numbers, and underscores.
- Variables can store strings, integers, and decimal values.

Valid examples:

```dmj
var studentName = "Alex"
var age = 18
var sensor_value = 42
```

---

## Example Program

```dmj
var name = "DMJScript"
var version = 1.0

show name
show version
```

Output:

```text
DMJScript
1.0
```
